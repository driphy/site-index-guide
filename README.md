# site-index-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，而是作为通用索引与导航资源进行维护。

## 项目简介

site-index-guide 是一个轻量级的 HTML 页面集合。所有页面均以静态 HTML 形式存放，可直接通过浏览器打开或部署到静态托管服务。仓库内容以索引和指南类页面为主，旨在提供结构化的信息呈现方式。

## 目录说明

```
site-index-guide/
├── index.html          # 主入口页面
├── pages/              # 归档页面目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── style.css
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库的主入口文件，通常包含所有归档页面的链接索引。
- `pages/`：存放所有独立 HTML 页面，每个文件代表一个归档单元。
- `assets/`：存放页面所需的公共静态资源，如样式表、脚本、图片等。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可单独打开或引用。
- 页面内容以信息归档与索引为主，不包含商业推广或特定网站链接。
- 归档内容可能不定期更新，但不会频繁变动。
- 页面命名建议使用英文小写，以连字符分隔，便于识别和链接。

## 维护说明

- 欢迎提交 Pull Request 新增或修正页面内容，但请保持内容风格一致。
- 新增页面请放置在 `pages/` 目录下，并在 `index.html` 中添加对应链接。
- 修改静态资源时请注意保持向后兼容性。
- 如果发现页面链接失效或内容错误，请提交 Issue 说明。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `index.html` 即可浏览所有归档页面。
3. 也可将整个仓库部署到任何静态 Web 服务器或托管平台。
